# 💤 LazyVim

A starter template for [LazyVim](https://github.com/LazyVim/LazyVim).

Install:
```
brew install nvim
git clone git@github.com:haney-oliver/nvim-config.git ~/.config/nvim
```
Use nvim as your default terminal in tools like `k9s` or `kubectl`:
```
echo 'export TERMINAL=nvim' >> ~/.zshrc
echo 'export DEFAULT_TERMINAL=nvim' >> ~/.zshrc
```
